# JavaScript/TypeScript SDK (base, generated)

Это base TypeScript/JavaScript SDK для Flagent API, **сгенерированное из OpenAPI** (использует `axios`).

Если вам нужен более удобный API (кэширование, удобные хелперы) — используйте `javascript-enhanced/`.

## Сборка

```bash
npm install
npm run build
```

## Установка

Если пакет опубликован:

```bash
npm install @flagent/client@0.1.4 --save
```

Локально (не рекомендуется для production):

```bash
npm install PATH_TO_GENERATED_PACKAGE --save
```
