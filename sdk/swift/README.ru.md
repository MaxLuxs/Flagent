# Swift SDK (base, generated)

Это base Swift SDK для Flagent API, **сгенерированное из OpenAPI** (Swift 5 generator).

Если вам нужен более удобный API (кэширование, higher-level abstraction) — используйте `swift-enhanced/`.

## Установка

### Carthage

```bash
carthage update
```

### CocoaPods

```bash
pod install
```
