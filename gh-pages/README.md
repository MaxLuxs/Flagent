# GitHub Pages — исходники сайта

Этот каталог публикуется на **https://maxluxs.github.io/Flagent/** через workflow [.github/workflows/pages.yml](../.github/workflows/pages.yml).

## Что делать

1. **Добавь сюда контент** — HTML, Markdown, статику. Сейчас здесь только заглушка.
2. **Если нужна текущая документация** — скопируй нужное из папки `docs/` (getting-started.md, api/, architecture/, styles и т.д.).
3. **Настройка в GitHub:** Repo → Settings → Pages → Source: **GitHub Actions** (workflow уже настроен).

При пуше в `main` с изменениями в `gh-pages/**` сайт обновится автоматически.
