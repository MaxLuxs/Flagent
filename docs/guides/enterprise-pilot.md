# Paid pilot (Enterprise)

A paid pilot lets you run Flagent Enterprise for a fixed period with a defined scope, then decide whether to move to an annual contract.

## Duration and scope

- **Duration:** 1–3 months (agreed up front).
- **Scope:** One environment, up to an agreed number of flags/experiments, single tenant (multi-tenant can be discussed for larger pilots).

## What we do

- Deploy Flagent Enterprise on-premise or in your cloud.
- Integration support: connect your auth (SSO), CI, or SDKs.
- Migrations from an existing DB or config if needed.
- Basic monitoring and health-check setup.
- Regular sync calls to track progress and questions.

## What you get

- Enterprise build: multi-tenancy, SSO, RBAC, Smart Rollout, Anomaly Detection, audit log, crash-by-flag analytics, Slack notifications.
- Support channel (email/chat).
- Documentation and best-practice guidance.
